Quick Start¶

Assuming that you have already attached and installed your Open-Channel SSD and running Ubuntu Linux (16.04) here is what you need to take liblightnvm for a spin:

echo "deb https://dl.bintray.com/openchannelssd/debs xenial main" | sudo tee -a /etc/apt/sources.list
sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 379CE192D401AB61
sudo apt-get update
sudo apt-get install liblightnvm

Hello NVM Device¶

Include the liblightnvm header in your C/C++ source:

#include <stdio.h>
#include <liblightnvm.h>

int main(int argc, char **argv)
{
	NVM_DEV dev = nvm_dev_open("/dev/nvme0n1");
	if (!dev) {
		perror("nvm_dev_open");
		return 1;
	}
	nvm_dev_pr(dev);
	nvm_dev_close(dev);

	return 0;
}

Then compile it using your e.g. gcc:

gcc hw.c -llightnvm -o hw

Tip

liblightnvm is also available as a static library. So you can add /usr/lib/liblightnvm.a to your compiler target instead of linking dynamically.

Now, go ahead and run it:

chmod +x hw
./hw

Interact via CLI¶

Check that it works by querying your device for information:

nvm_dev info /dev/nvme0n1

Which should output information similar to:
