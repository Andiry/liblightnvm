Quick StartÂ¶

Assuming that you have already attached and installed your Open-Channel SSD and running Ubuntu Linux (16.04) here is what you need to take liblightnvm for a spin:

echo "deb https://dl.bintray.com/openchannelssd/debs xenial main" | sudo tee -a /etc/apt/sources.list
sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 379CE192D401AB61
sudo apt-get update
sudo apt-get install liblightnvm

Check that it works by querying your device for information:

nvm_info /dev/nvme0n1

Which should output information similar to:

** Device information  -- nvm_dev_pr **

Tip

If the above does not suffice to get you started then have a look at the Prerequisites for detailed information about what you need to setup an environment.

With the basics in place, explore the C API, the Command-line Interface, and the Python Module.
